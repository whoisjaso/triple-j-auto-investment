<task type="auto">
  <name>Task 1: Admin Behavior Intelligence panel</name>
  <files>triple-j-auto-investment-main/components/admin/AdminBehaviorPanel.tsx</files>
  <action>
Create an admin-only panel component that displays three data sections. This is read-only, no export functionality.

**Data fetching (on mount, no auto-refresh):**
Run three Supabase queries in parallel via Promise.all:

1. **Top Viewed Vehicles (last 7 days):**
```typescript
supabase.from('vehicle_view_counts')
  .select('vehicle_id, views_7d, unique_sessions_7d')
  .gt('views_7d', 0)
  .order('views_7d', { ascending: false })
  .limit(10)
```
Then join with vehicles from the store to get year/make/model.

2. **Recent Session Activity:**
```typescript
supabase.from('session_events')
  .select('session_id, created_at, event_type, vehicle_id, page_path')
  .order('created_at', { ascending: false })
  .limit(100)
```
Group by session_id to show a feed of recent sessions with event counts.

3. **Lead Attribution Breakdown:**
```typescript
supabase.from('leads')
  .select('utm_source, utm_medium, device_type, page_path, created_at')
  .not('session_id', 'is', null)
  .order('created_at', { ascending: false })
  .limit(200)
```

**Rendering three sections:**

Section 1 -- "Top Viewed Vehicles":
- Table/list with columns: Vehicle (Year Make Model), Views (7d), Unique Visitors
- Each row shows the vehicle name and counts
- If no data, show "No tracking data yet" message

Section 2 -- "Recent Sessions":
- Group the 100 most recent events by session_id
- Show each session as: session ID (truncated first 8 chars), event count, most recent timestamp, pages visited
- Limit display to 20 most recent unique sessions
- If no data, show "No sessions recorded yet"

Section 3 -- "Attribution Breakdown":
- Three mini tables/lists:
  a. By Device: count leads per device_type (mobile/tablet/desktop), show as horizontal bar or simple count
  b. By Source: count leads per utm_source (group nulls as "Direct"), show top 5
  c. By Page: count leads per page_path, show top 5
- If no attributed leads, show "No attributed leads yet"

**Styling:**
- Use admin dashboard styling patterns (already in Dashboard.tsx): bg-black, border-white/[0.06], text-white headings, text-gray-400 body text
- Section headers with lucide icons: Activity for Top Viewed, Radio for Sessions, Target for Attribution
- Compact tables with text-xs for data density
- Loading skeleton while fetching (simple pulse animation)

**Props:** None (self-contained, gets vehicles from useStore).
