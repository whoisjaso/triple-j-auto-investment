    **Step 1: Establish canonical patterns.**

    Before editing any files, define these standard patterns that ALL pages must follow:

    **Spacing scale:**
    - **Page horizontal padding:** `px-4 md:px-6` (base for all page content wrappers)
    - **Section vertical padding (standard):** `py-16 md:py-24` (most content sections)
    - **Section vertical padding (large/hero):** `py-20 md:py-32` (hero sections, major CTAs)
    - **Card/panel padding:** `p-6 md:p-8` (all info cards, content panels)
    - **Inner element padding:** `p-3` or `p-4` (inputs, small containers)
    - **Section gap inside:** `space-y-8` or `gap-8` (between items within a section)
    - **Max content width:** Keep existing `max-w-7xl` or `max-w-[1400px]` patterns per page

    **Button patterns (4 types):**
    1. **Primary CTA (gold):** `bg-tj-gold text-black font-bold py-4 px-8 text-xs uppercase tracking-[0.3em] hover:bg-white transition-colors`
    2. **Primary CTA large (hero):** Same as above but `py-5 px-12 md:py-6 md:px-16` for hero/full-width CTAs
    3. **Ghost/outline:** `border border-tj-gold text-tj-gold font-bold py-4 px-8 text-xs uppercase tracking-[0.3em] hover:bg-tj-gold hover:text-black transition-colors`
    4. **Text link:** `text-tj-gold text-xs uppercase tracking-[0.3em] hover:text-white transition-colors`
    5. **Danger (admin only):** `text-red-500 text-xs uppercase tracking-widest hover:text-red-400 transition-colors` (only in admin areas, no changes needed)

    **Key standardization rules:**
    - ALL tracking should be `tracking-[0.3em]` (not `tracking-[0.2em]` or `tracking-widest`)
    - ALL button font should be `text-xs` (not `text-sm` or `text-[10px]`)
    - ALL primary buttons should be `py-4 px-8` (not `py-3` or `py-5` or `py-6` unless it's the hero variant)

    **Inventory.tsx:**
    - Standardize page wrapper horizontal padding to `px-4 md:px-6`
    - Standardize any card padding in vehicle cards
    - Standardize the "I'm Interested" and other CTA buttons to canonical pattern
    - Vehicle card padding: if inconsistent, standardize to `p-6 md:p-8`

    **About.tsx:**

    - Standardize section padding
    - Standardize any CTA buttons

    **Finance.tsx:**
    - Standardize form submit button to primary CTA pattern
    - Standardize info panel/card padding
    - Standardize section padding

    **Important constraints:**
    - Do NOT change colors, border styles, or the overall design aesthetic
    - Do NOT change layout structures (grid layouts, flex arrangements)

    - Only modify: padding values, font sizes on buttons, tracking values on buttons
