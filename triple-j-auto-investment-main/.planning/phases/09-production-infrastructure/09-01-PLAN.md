<task type="checkpoint:human-action" gate="blocking">
  <name>Task 1: Create Resend account, add domain, get DNS records into Namecheap</name>
  <action>
    This task has a 48-hour lead time -- start it first.

    Steps:
    1. Go to https://resend.com and create a free account (100 emails/day free tier)
    2. In Resend Dashboard -> Domains -> Add Domain: enter `triplejautoinvestment.com`
    3. Resend will show DNS records to add (typically 1 SPF TXT record + 3 DKIM CNAME records)
    4. Copy ALL DNS records Resend provides
    5. Go to Namecheap -> Domain List -> triplejautoinvestment.com -> Advanced DNS
    6. Add each DNS record exactly as Resend specified:
       - SPF: TXT record at @ with the provided value
       - DKIM: CNAME records with the provided host/value pairs
    7. Back in Resend, click "Verify" -- it will show "Pending" (this is expected)
    8. Copy the Resend API key from Dashboard -> API Keys and save it securely (you will need it in Plan 03 for Edge Function secrets)
    9. DNS propagation takes up to 48 hours -- verification will complete automatically

    IMPORTANT: Do NOT delete any existing DNS records at Namecheap. Only ADD the new Resend records alongside whatever already exists.
  </action>
  <verify>
    - Resend dashboard shows the domain `triplejautoinvestment.com` with status "Pending" or "Verified"
    - Namecheap Advanced DNS shows the new SPF and DKIM records
    - You have the Resend API key saved securely
  </verify>
  <done>Resend account exists, domain DNS records are submitted, API key is saved. Verification will complete within 48 hours.</done>
  <resume-signal>Confirm: "Resend DNS records added" or describe issues</resume-signal>
</task>
