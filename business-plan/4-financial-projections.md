## Break-Even Analysis

### Fixed Costs (Monthly)
```
Platform: $200
Salaries: $10,000
Insurance: $300
Marketing: $4,000
Operations: $1,000
Total: $15,500/month
```

### Contribution Margin per Vehicle
```
Sale Price: $45,000
COGS: $23,000
Variable Marketing: $2,083
Contribution Margin: $19,917
```

### Break-Even Units
```
Break-Even = Fixed Costs / Contribution Margin
= $15,500 / $19,917
= 0.78 vehicles/month
= ~9.4 vehicles/year
```

**Conclusion:** Need to sell just 10 vehicles/year to break even. Anything above 10 is pure profit.
