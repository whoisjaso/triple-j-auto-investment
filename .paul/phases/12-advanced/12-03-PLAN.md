# Plan 12-03: Rental Renewal + Mobile Polish

## Scope

Final plan for Phase 12. Implements rental renewal flow from AgreementTracker,
mobile UX polish across document pages, and remaining tests.

## Tasks

### Task 1: Renewal Button in AgreementTracker
- Add "Renew" button on completed rental agreements
- Only visible when `document_type === 'rental'` AND `status === 'completed'`
- Navigates to `/admin/documents/rental?renew={agreementId}`
- Uses lucide `RotateCcw` icon, styled to match existing action buttons

**File:** `src/app/admin/documents/agreements/AgreementTracker.tsx`

### Task 2: Renewal Pre-fill in DocumentEditor
- Rental page reads `?renew` searchParam, passes to DocumentEditor
- DocumentEditor accepts optional `renewAgreementId` prop
- On mount, fetch `/api/documents/agreements/{id}` → get `completed_link`
- Decode via `decodeCompletedLinkFromUrl()` → extract renter + vehicle + rate data
- Pre-fill `rentalData` state, merge dealer data (dd) + customer data (cd)
- Reset dates (start=today, end=blank), mileage, fuel levels
- Show loading state during fetch, error banner on failure

**Files:**
- `src/app/admin/documents/rental/page.tsx` (pass searchParam)
- `src/components/documents/DocumentEditor.tsx` (renewal logic)

### Task 3: Mobile UX Polish
- All tap targets ≥ 48px on document pages
- Input font-size ≥ 16px (prevents iOS auto-zoom)
- Smooth transitions on buttons and interactive elements
- Focus on AgreementTracker, DocumentEditor, CustomerWizard

**Files:** Various document components

### Task 4: Tests + Playwright Verification
- Unit test: renewal pre-fill data mapping
- Playwright: verify renewal flow desktop + mobile
- Playwright: verify mobile tap targets and input sizing

**Files:** `src/__tests__/rentalRenewal.test.ts`

## Acceptance Criteria

| Criterion | Description |
|-----------|-------------|
| Renew button visible | Completed rental agreements show Renew button |
| Renewal pre-fills data | Clicking Renew → rental page loads with pre-filled customer+vehicle data |
| Dates reset | Start date = today, end date = blank on renewal |
| Mobile tap targets | All interactive elements ≥ 48px touch target |
| No iOS zoom | Input font-size ≥ 16px prevents auto-zoom |
| Tests pass | All existing + new tests pass |
| Playwright verified | Screenshots confirm UX on mobile + desktop |
