<objective>
## Goal
Build lead management page, dashboard statistics, and a discoverable admin login link — completing the admin core for v0.1.

## Purpose
Jason needs to view and manage customer inquiries from the website. The dashboard needs at-a-glance stats (vehicle count, lead count, recent leads). And there must be a way for Jason to reach the admin login from the public site without exposing it prominently to customers.

## Output
- `/admin/leads` page: table of leads with status badges, status toggle (New → Contacted → Closed), filter by status
- `/admin` dashboard: live stats cards (total vehicles, total leads, new leads count) + recent leads preview
- Footer: subtle "Dealer Login" link in compliance bar (low-profile, next to dealer license)
- Mock leads fallback for development without Supabase
</objective>
