<objective>
## Goal
Build the contact page, financing inquiry page, and lead submission infrastructure — completing the conversion funnel from vehicle browsing to dealer contact.

## Purpose
Every page on the site drives toward one action: contact the dealer. The VehicleInquiryButton already links to `/contact?vehicle={name}`, the Navbar already links to `/contact` and `/financing`, but neither page exists. This phase completes the funnel so every CTA resolves to a working form that captures the lead.

## Output
- `/contact` page with lead capture form (pre-fills vehicle from query param)
- `/financing` page with BHPH-focused financing inquiry form
- Server action for lead submission (Supabase when connected, graceful fallback when not)
- Reusable ContactForm client component
- Both pages match luxury aesthetic (black bg, gold accents, cream headings)
</objective>

    - Validate: name required (min 2 chars), phone required (min 10 digits)

    **Create `src/components/leads/ContactForm.tsx`:**
    - "use client" component
    - Props: `{ source: LeadSource; vehicleName?: string; showVehicleField?: boolean; showDownPayment?: boolean; showBudget?: boolean }`
    - Uses useActionState (React 19) with submitLead server action
    - Form fields (all inside a single <form>):
      - Name: text input (required)
      - Phone: tel input (required)
      - Email: email input (optional)
      - Vehicle of Interest: text input (shown when showVehicleField=true, pre-filled with vehicleName)
      - Down Payment Budget: text input (shown when showDownPayment=true, placeholder "e.g., $500")
      - Message: textarea (optional, 4 rows)
      - Hidden input: source (LeadSource value)
    - States:
      - idle: form visible, submit button enabled
      - pending: submit button disabled, shows "Sending..." with loading indicator
      - success: form hidden, success message shown ("Thank you! We'll be in touch soon.")
      - error: form visible, error message shown at top
    - Input styling:
      - bg-transparent border-b border-white/10 focus:border-tj-gold/30
      - text-tj-cream text-sm min-h-[44px]
      - Label: font-accent text-[9px] uppercase tracking-[0.2em] text-white/25
    - Submit button:
      - bg-tj-gold/90 hover:bg-tj-gold text-black font-accent uppercase tracking-[0.2em]
      - Full width, min-h-[52px], rounded-sm
    - Inline validation: highlight invalid fields on submit, focus first error
    - Success state: green checkmark icon + message + "Call us now" link
