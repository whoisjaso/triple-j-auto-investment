<objective>
## Goal
Build a responsive navbar and footer that wrap the cinematic homepage and all future pages, completing the layout shell for Phase 3.

## Purpose
Every page needs navigation and dealer compliance info. The navbar is the first brand touchpoint on non-homepage pages and must project the same luxury authority as the scroll animations. The footer grounds the experience with real-world trust signals (license, address, phone). Together they complete the navigable layout shell that Phase 4+ pages will slot into.

## Output
- `src/components/layout/Navbar.tsx` — Responsive navbar with gold crest, nav links, mobile menu
- `src/components/layout/Footer.tsx` — Dealer compliance footer with contact, license, links
- Updated `src/app/layout.tsx` — Integrates navbar + footer around children
- Updated `src/app/globals.css` — Any additional CSS for navbar/footer transitions
</objective>
