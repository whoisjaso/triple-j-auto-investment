## Task 2: Enhance schema markup + add FinancialProduct/Service schemas + hreflang

**Files:** `triple-j-auto-investment-main/index.html`, `triple-j-auto-investment-main/components/SEO.tsx`

### 2A: Enhance existing AutoDealer schema in index.html

Add to the AutoDealer JSON-LD block:
- `"knowsLanguage": ["en", "es"]`
- `"additionalType": "https://schema.org/AutoRental"` (signals rental business line)
- `"sameAs"` array with business profile URLs (use placeholder URLs that owner can update):
  ```json
  "sameAs": [
    "https://www.facebook.com/triplejautoinvestment",
    "https://www.instagram.com/triplejautoinvestment"
  ]
  ```
- `"hasOfferCatalog"` -- add a third entry for vehicle rentals

### 2B: Add FinancialProduct JSON-LD for Buy Here Pay Here

Add a new `<script type="application/ld+json">` block:
```json
{
  "@context": "https://schema.org",
  "@type": "FinancialProduct",
  "name": "In-House Auto Financing",
  "description": "Buy Here Pay Here in-house auto financing for Houston families. No bank approval required. Down payments starting at $1,000 on vehicles in the $3,000-$8,000 range.",
  "provider": {
    "@type": "AutoDealer",
    "name": "Triple J Auto Investment",
    "url": "https://triplejautoinvestment.com"
  },
  "areaServed": {
    "@type": "City",
    "name": "Houston",
    "containedInPlace": {
      "@type": "State",
      "name": "Texas"
    }
  },
  "category": "Auto Financing",
  "url": "https://triplejautoinvestment.com/finance"
}
```
