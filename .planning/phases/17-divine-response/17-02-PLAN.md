**useDivineChat.ts** -- Chat state management hook:

1. Import `sendChatMessage, ChatMessage, VehicleContext, loadChatHistory, saveChatHistory` from `divineChatService`.
2. Import `identifyProfile, ProfileType` from `chatProfiles`.
3. Import `getSessionId` from `trackingService` (reuse existing session ID).
4. Hook signature: `useDivineChat(vehicleContext: VehicleContext, vehicleId: string, language: 'en' | 'es')`
5. State:
   - `messages: ChatMessage[]` -- initialized from `loadChatHistory(vehicleId)`
   - `isStreaming: boolean` -- true while response is being received
   - `profile: ProfileType` -- starts as 'unidentified'
   - `error: string | null` -- error message or null
6. `abortRef = useRef<AbortController | null>(null)` for canceling in-flight requests.
7. `messagesEndRef = useRef<HTMLDivElement>(null)` for scroll-to-bottom (exported for ChatWidget to attach).
8. Auto-scroll: `useEffect` that calls `messagesEndRef.current?.scrollIntoView({ behavior: 'smooth' })` when messages change.
9. Auto-save: `useEffect` that calls `saveChatHistory(vehicleId, messages)` when messages change (debounce not needed -- localStorage is synchronous and fast).
10. `sendMessage` callback (wrapped in `useCallback`):
    a. Set error to null.
    b. Create user ChatMessage with `crypto.randomUUID()` id, role 'user', text, timestamp Date.now().
    c. Add to messages.
    d. Run `identifyProfile()` on all user message texts so far (including new one). Update profile state.
    e. Create placeholder model ChatMessage with empty text.
    f. Set isStreaming true.
    g. Try: create AbortController, fetch via `sendChatMessage()` with abort signal passed as second arg.
    h. If response not ok: throw Error('Service unavailable').
    i. Read response.body with `getReader()` + TextDecoder. Loop: read chunks, accumulate text, update placeholder message text via setMessages map.
    j. Catch: if not AbortError, set error to bilingual fallback message (use language param to pick en/es). Remove placeholder model message.
    k. Finally: set isStreaming false.
11. `clearChat` callback: clear messages state, call `clearChatHistory(vehicleId)`, reset profile to 'unidentified'.
12. Return: `{ messages, sendMessage, clearChat, isStreaming, profile, error, messagesEndRef }`
