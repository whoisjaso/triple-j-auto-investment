**OwnerValueTracker.tsx** -- Value chart + cost-per-day:
- Props: { registration: Registration }
- Import { LineChart, Line, XAxis, YAxis, ResponsiveContainer, Tooltip } from 'recharts'.
- Import { estimateMarketValue } from '../../services/marketEstimateService'.
- Import useLanguage for bilingual labels.

**buildValueHistory function:**
- Takes purchasePrice, purchaseDate (string), vehicleYear (number), mileageAtPurchase (number | undefined).
- Generates monthly data points from purchase date to current month.
- For each month: compute months elapsed, estimate mileage progression at ~1000 miles/month (use mileageAtPurchase as base, default to 80000 if undefined), compute estimateMarketValue(purchasePrice, vehicleYear, estimatedMileage).
- Returns Array<{ month: string; value: number }>.
- Month label format: short month + 2-digit year (e.g., "Mar '26").

**computeValueMetrics function:**
- Computes currentValue from estimateMarketValue with current age/mileage estimate.
- Computes costPerDay = purchasePrice / Math.max(1, daysOwned) where daysOwned = floor((now - purchaseDate) / 86400000). Guards against division by zero per research pitfall.
- Returns { currentValue, costPerDay, daysOwned }.

**Render:**
- Card with same styling (p-6 md:p-8 bg-black/40 border border-tj-gold/10 rounded-lg).
- Section title: t.ownerPortal.valueTracker.
- Stats row (flex, gap-4, wrap on mobile):
  - Current estimated value: large text-white font-serif, formatted as "$X,XXX"
  - Cost per day: text-tj-gold, formatted as "$X.XX/day"
  - Purchase price: text-gray-400, formatted as "$X,XXX"
- Mini line chart using Recharts:
  - ResponsiveContainer width="100%" height={120} (CRITICAL: fixed pixel height, NOT percentage -- per research pitfall).
  - LineChart with margin { top: 4, right: 4, bottom: 0, left: 0 }.
  - XAxis: dataKey="month", tick fontSize 10, fill '#9CA3AF', tickLine false.
  - YAxis: hide (value shown in stats above).
  - Tooltip: formatter shows "$X,XXX" with 'Est. Value' label. Content style: background '#0e1b16', border '1px solid #d4af3730', borderRadius 4. Label style: color '#9CA3AF', fontSize 10.
  - Line: type="monotone", dataKey="value", stroke="#d4af37" (tj-gold), strokeWidth 1.5, dot={false}.
- Below chart: small text-gray-400 label "Value Over Time".
- If purchaseDate is very recent (< 2 months), show just the current stats without the chart (not enough data points for a meaningful line).
