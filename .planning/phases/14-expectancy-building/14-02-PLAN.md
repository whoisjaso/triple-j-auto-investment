  <action>
    Add two new exported functions to geminiService.ts, following the exact same pattern as `generateVehicleDescription`:

    **generateIdentityHeadline** -- Generates a bilingual identity-first headline.
    - Parameters: `make: string, model: string, year: number, bodyType?: string, diagnostics: string[]`
    - Returns: `Promise<{ en: string; es: string }>`
    - Prompt rules:
      1. Format: "[Identity Label] | [2-3 punchy descriptive words]."
      2. Lead with WHO this car is for, not what it is
      3. Maximum 15 words total
      4. Warm, aspirational but honest tone
      5. If diagnostics mention issues, acknowledge in word choice (e.g., "Project-Ready" instead of "Family-Ready")
      6. Examples: "Family-Ready Sedan | Reliable. Clean. Ready for Your Next Chapter.", "Weekend Warrior | Tough. Capable. Built for Adventure.", "Daily Driver | Efficient. Dependable. Your Everyday Companion.", "First Car Ready | Safe. Simple. Perfect for New Drivers."
    - Ask Gemini to return JSON: `{"en": "English headline", "es": "Spanish headline"}`
    - Parse JSON response. On parse failure, fall back to `{ en: "${year} ${make} ${model}", es: "${year} ${make} ${model}" }`
    - On client error (no API key), return the same fallback
    - Use `gemini-2.5-flash` model with `thinkingBudget: 1024`
    - Strip any markdown code fences from the response before JSON.parse (Gemini sometimes wraps JSON in ```json blocks)

    **generateVehicleStory** -- Generates a bilingual honest vehicle story.
    - Parameters: `make: string, model: string, year: number, mileage: number, diagnostics: string[], description: string`
    - Returns: `Promise<{ en: string; es: string }>`
    - Prompt rules:
      1. Write a 3-5 sentence honest story about this vehicle
      2. Include: what type of owner this car is ideal for, its strengths, and any condition notes
      3. If diagnostics list any issues, disclose them honestly but frame them as known and transparent ("We noticed X and want you to know upfront")
      4. If no issues, highlight reliability and value
      5. Tone: warm, transparent, trustworthy. Like a neighbor giving honest advice.
      6. DO NOT fabricate history (no "one-owner" claims, no "garage-kept" unless in diagnostics)
      7. End with a forward-looking statement about the vehicle's next chapter
    - Ask Gemini to return JSON: `{"en": "English story", "es": "Spanish story"}`
    - Same error handling and model config as generateIdentityHeadline
    - Strip markdown code fences before JSON.parse

    IMPORTANT: Both functions must handle the common Gemini response pattern where JSON is wrapped in ```json ... ``` code fences. Add a helper at module level:

    ```typescript

      return text.replace(/```json\s*/g, '').replace(/```\s*/g, '').trim();
    }
    ```
